# 忆刻 (YiKe) — UI/UX 设计文档

## 文档概述

本文档是忆刻产品的 UI/UX 设计提纲，涵盖设计体系、页面设计、组件规范等内容。详细设计稿请参考各版本 UI/UX 文档。

---

## 1. 设计概述

### 1.1 产品定位

基于艾宾浩斯遗忘曲线的学习任务规划应用，面向学生、职场人、终身学习者。

### 1.2 设计目标

- **零门槛**：输入即规划，用户无需理解记忆曲线理论
- **专注高效**：简洁界面，聚焦核心任务
- **温暖友好**：专业但不冰冷，激励用户持续学习

## 3. 页面设计

### 3.1 首页 — 今日任务

- 顶部日期选择器 + 待复习任务列表 + 底部浮动按钮
- 任务卡片显示：标题 | 标签 | 第 N 次复习 | 勾选框
- 左滑：跳过；右滑：完成
- 空状态：显示鼓励语

### 3.2 录入页

- 标题输入框 + 备注区域 + 标签选择 + 确认按钮
- 确认后 Toast 提示
- 支持连续录入模式

### 3.3 日历视图（v2.0）

- 月历 + 下方任务列表
- 日期下方显示任务状态圆点
- 颜色区分：绿色（已完成）/ 红色（逾期）/ 蓝色（待复习）

### 3.4 统计面板（v2.0）

- 连续打卡天数卡片
- 本周 / 本月完成率（环形进度）
- 标签分布饼图
- 学习记录列表

### 3.5 数据导出（v2.0）

- 格式选择：JSON / CSV
- 内容选择：学习内容 / 复习任务
- 数据预览 + 导出按钮

### 3.6 设置页

- 提醒设置（提醒时间、免打扰时段）
- 复习设置（复习间隔）
- 数据管理（数据导出、备份）
- 外观设置（主题模式 v2.2）
- 关于

## 4. 组件规范

### 4.1 按钮

| 类型 | 样式 | 使用场景 |
|------|------|----------|
| Primary | 背景 CTA 橙色，白色文字 | 主要行动（录入、完成） |
| Secondary | 背景 Primary | 次要行动 |
| Ghost | 背景透明，文字 Primary | 文字按钮 |
| Danger | 背景 Error | 删除、跳过 |

### 4.2 卡片

| 类型 | 样式 | 使用场景 |
|------|------|----------|
| Glass Card | 毛玻璃效果 | 进度展示、统计 |
| Task Card | 白色背景 + 状态边条 | 复习任务 |
| Input Card | 浅灰背景 + 边框 | 输入区域 |

### 4.3 标签

| 类型 | 样式 | 使用场景 |
|------|------|----------|
| Category Tag | 圆角 8px，背景 Primary/20% | 学习分类 |
| Status Tag | 圆角 full，颜色根据状态 | 任务状态 |

### 4.4 表单元素

| 组件 | 规范 |
|------|------|
| TextField | 高度 48px，圆角 12px，标签浮动 |
| Checkbox | 24x24px，圆角 6px |
| Switch | 51x31px，Primary 开启态 |

### 4.5 毛玻璃组件（GlassCard）

| 元素 | 浅色模式 | 深色模式 |
|------|----------|----------|
| 背景色 | `rgba(255,255,255,0.80)` | `rgba(22,33,62,0.80)` |
| 边框色 | `rgba(255,255,255,0.40)` | `rgba(255,255,255,0.10)` |
| 模糊度 | 14px | 14px |
| 圆角 | 16px | 16px |

## 6. 动画规范

### 6.1 过渡时长

| 场景 | 时长 | 缓动函数 |
|------|------|----------|
| 按钮 hover | 200ms | ease-out |
| 页面切换 | 300ms | ease-in-out |
| 卡片展开 | 250ms | ease-out |
| 弹窗出现 | 200ms | ease-out |
| 勾选完成 | 150ms | ease-in |

### 6.2 主题切换动画

- 背景色渐变：300ms
- 卡片轻微缩放：100ms

## 7. 空状态与异常状态

### 7.1 空状态

| 场景 | 文案 |
|------|------|
| 首页无任务 | "今天没有复习任务，继续保持！" |
| 日历无数据 | "本月暂无学习记录" |
| 统计无数据 | "还没有学习数据" |

### 7.2 异常状态

| 场景 | 状态 | 操作 |
|------|------|------|
| 通知权限被拒绝 | 弹窗引导 | 跳转系统设置 |
| 任务堆积（>20条） | 黄色警告条 | 优先完成红色逾期任务 |
| 数据库异常 | 错误页 | "数据加载失败，请重启应用" |

---

## 8. 无障碍设计

### 8.1 颜色对比度

- 正文文字：与背景对比度 ≥ 4.5:1
- 大字标题：与背景对比度 ≥ 3:1
- 按钮文字：与按钮背景对比度 ≥ 4.5:1

### 8.2 触摸目标

- 最小触摸区域：44x44px
- 列表项最小高度：48px

### 8.3 屏幕阅读器

- 所有图片添加语义标签
- 按钮/输入框有明确标签
- 状态变化有语音提示
