# 忆刻 (YiKe) — 文档索引

## 项目概述

基于艾宾浩斯遗忘曲线的学习任务规划应用，支持 Android / iOS 移动端与 Windows 桌面端。

## 技术栈

- Flutter 3.11+ / Riverpod 2.5+ / Drift / GoRouter
- 学习指南：flutter_markdown
- 桌面端：window_manager + tray_manager
- 同步：UDP 广播 + HTTP 传输（端口 19876/19877）

### 内容同步规则
- **学习指南**：以 `docs/prd/忆刻学习指南.md` 为唯一源，v3 开发时同步到 `assets/markdown/learning_guide.md`（需创建目录并配置 pubspec.yaml）
- **注意**：`assets/markdown/learning_guide.md` 为生成文件，不手工编辑，避免双向内容漂移
- **同步脚本**：使用 `tool/sync_learning_guide.dart` 一键同步学习指南到 assets
