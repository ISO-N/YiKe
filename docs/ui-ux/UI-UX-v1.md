## 1. 设计概述

### 1.1 产品定位
基于艾宾浩斯遗忘曲线的学习任务规划应用，面向学生、职场人、终身学习者。

### 1.2 设计目标
- **零门槛**：输入即规划，用户无需理解记忆曲线理论
- **专注高效**：简洁界面，聚焦核心任务
- **温暖友好**：专业但不冰冷，激励用户持续学习

## 2. 设计体系

### 2.1 视觉风格

| 属性 | 值 |
|------|-----|
| **风格** | Glassmorphism（毛玻璃） |
| **关键词** | 透明、模糊背景、层次感、质感、现代 |
| **适用场景** | 卡片、弹窗、导航栏、背景层叠 |

### 2.6 阴影与效果

**毛玻璃效果（Glass Card）：**
```css

border: 1px solid rgba(255, 255, 255, 0.4);
box-shadow: 0 4px 24px rgba(13, 148, 136, 0.08);
```

```css

```


## 7. 空状态与异常状态

### 7.1 空状态设计

| 场景 | 插图 | 文案 |
|------|------|------|
| 首页无任务 | 空白 + 勾选图标 | "今天没有复习任务，继续保持！🎉" |
| 录入页无标签 | 空白 + 标签图标 | "还没有标签，创建一个吧" |
| 日历无数据 | 空白 + 日历图标 | "本月暂无学习记录" |

**空状态样式：**
- 插图尺寸：120x120px
- 文案颜色：Text Secondary (`#64748B`)
- 垂直居中，水平居中

### 7.2 异常状态设计

| 场景 | 状态 | 操作 |
|------|------|------|
| 通知权限被拒绝 | 弹窗引导 | "开启通知，确保及时收到复习提醒" → 跳转系统设置 |
| 任务堆积（>20条） | 黄色警告条 | "今日任务较多，建议优先完成红色逾期任务" |
| 数据库异常 | 错误页 | "数据加载失败，请重启应用" + 重试按钮 |

### 7.3 权限引导

**通知权限引导流程：**
```
1. 用户首次进入应用
2. 检测通知权限未授权
3. 弹窗说明："开启通知后，我们会在复习时间提醒你"
4. 用户点击"去开启" → 跳转系统通知设置
5. 用户拒绝 → 记录状态，后续不再频繁弹窗（可设置页再次引导）
```

## 9. 无障碍规范

### 9.1 颜色对比度
- 正文文字：与背景对比度 ≥ 4.5:1
- 大字标题：与背景对比度 ≥ 3:1
- 按钮文字：与按钮背景对比度 ≥ 4.5:1

### 9.2 触摸目标
- 最小触摸区域：44x44px
- 列表项最小高度：48px

### 9.3 屏幕阅读器
- 所有图片添加语义标签
- 按钮/输入框有明确标签
- 状态变化有语音提示

### 8.4 键盘导航
- Tab 顺序与视觉顺序一致
- 焦点状态可见（Primary 颜色边框）
- 支持系统返回键
